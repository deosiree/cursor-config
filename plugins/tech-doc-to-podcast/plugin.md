# 技术文档转双人播客插件

将一份技术类Markdown文档转换为双人播客脚本，包含按问题拆分的Markdown格式。

## 功能特性

### 1. 文档分析
- 分析技术文档结构和内容
- 提取关键概念和知识点
- 识别适合播客讨论的话题

### 2. 播客脚本生成
- 生成双人对话格式的播客脚本
- 设计主持人和嘉宾的角色
- 创建自然的对话流程

### 3. 问题拆分
- 按问题拆分Markdown文档
- 创建结构化的问答格式
- 支持深度讨论和扩展

## 触发方式

### 推荐用法
- `将这个技术文档转换为播客`
- `生成双人播客脚本`
- `按问题拆分技术文档`
