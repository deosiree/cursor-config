# 文件优化点批量分析插件

用于批量分析指定前端文件（重点面向Vue/JS/TS）中的优化点、潜在问题和改进建议。

## 功能特性

### 1. 代码质量分析
- 分析代码结构和组织
- 检查代码重复和冗余
- 识别潜在的性能问题

### 2. 最佳实践检查
- 检查编码规范和最佳实践
- 分析代码可读性和可维护性
- 识别安全问题和漏洞

### 3. 优化建议
- 提供具体的优化建议
- 建议重构方案
- 推荐最佳实践

## 触发方式

### 推荐用法
- `@file-check 分析这个文件的优化点`
- `@file-check 批量分析这些文件的问题`
- `@file-check 检查代码质量和最佳实践`

## 分析步骤

### 1. 文件收集
- 收集需要分析的文件
- 确定分析范围和重点
- 准备分析环境

### 2. 代码分析
- 分析代码结构和组织
- 检查编码规范和最佳实践
- 识别潜在问题和优化点

### 3. 报告生成
- 生成详细的分析报告
- 提供具体的优化建议
- 建议重构方案

## 使用场景

### 场景1：代码质量检查
- 检查代码质量和最佳实践
- 识别潜在的性能问题
- 分析代码可维护性

### 场景2：优化点分析
- 分析代码优化点
- 提供重构建议
- 推荐最佳实践

### 场景3：批量文件分析
- 批量分析多个文件
- 生成综合报告
- 提供整体优化建议
